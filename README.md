# Customer Churn Analysis and Prediction

This project focuses on analyzing customer churn data from a telecommunications company and building machine learning models to predict customer churn.

## Project Objective

The main goal of this project is to:

- Analyze customer behavior
- Identify customers likely to churn
- Build predictive machine learning models
- Provide insights to improve customer retention

# Dataset

Dataset Used:
- Telco Customer Churn Dataset

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
 
#  Tasks Performed

## Task 1 — Data Preparation

- Data loading
- Missing value handling
- Label encoding
- Train-test split

## Task 2 — Exploratory Data Analysis

- Churn rate visualization
- Gender distribution analysis
- Contract type analysis
- Payment method analysis

## Task 3 — Customer Segmentation

- Customer segmentation based on:
  - Tenure
  - Monthly charges
  - Contract type

## Task 4 — Churn Prediction Model

Machine Learning Models Used:
- Logistic Regression

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1 Score

## Task 5 — Model Evaluation

- Confusion Matrix
- ROC Curve
- AUC Score

#  Sample Visualization

- Churn Count Plot
- ROC Curve
- Customer Distribution Graphs

# Machine Learning Workflow

1. Load Dataset
2. Clean Data
3. Encode Categorical Data
4. Split Dataset
5. Train Model
6. Predict Results
7. Evaluate Performance

#  Results

- Successfully trained a churn prediction model
- Identified important customer churn patterns
- Visualized customer behavior using graphs

# Learning Outcomes

Through this project, I learned:

- Data preprocessing
- Exploratory data analysis
- Machine learning basics
- Model evaluation
- Customer churn prediction
